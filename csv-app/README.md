# CSV Graph Visualization (local)

Small Vite + React app to upload a CSV and visualize columns as a chart.

Quick start

1. cd into the app folder

```bash
cd csv-app
npm install
npm run dev
```

2. Open the dev server shown by Vite (usually http://localhost:5173)

Usage

- Drop a CSV file, choose file or paste CSV text into the upload area.
- Select X and Y columns and choose chart type (Line or Bar).

Dependencies: React, PapaParse, Chart.js, react-chartjs-2, Vite
